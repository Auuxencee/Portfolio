Two main projects built during my engineering studies — both hands-on, documented, and deployable. More on GitHub.

Featured Project — 2024
AegisQuantum
Problem
Most cryptographic libraries don't expose post-quantum primitives in an accessible way. Classical RSA and ECDH will be broken by large quantum computers — the transition needs tooling that developers can actually use today.
Approach
Implemented Kyber-1024 (key encapsulation) and Dilithium (digital signatures) from scratch in Python, following NIST PQC Round 3 specifications. Built a clean API surface for hybrid classical/quantum key exchange.
Python Kyber-1024 Dilithium NIST PQC Cryptography

2024
SOC Training Laboratory
Problem
Learning defensive security from certifications alone doesn't build operational instinct. Real SOC skills require detecting actual attack patterns in live infrastructure.
Approach
Built a self-contained SOC environment using Docker Compose — Wazuh Manager, Indexer, and Dashboard. Progressive difficulty: start with basic detection, escalate to simulated real attacks, then automate response with custom rules and playbooks.
Wazuh Docker OpenSearch SIEM Incident Response

AI Security Assistant
Problem
Security workflows involve repetitive triage work — reading logs, writing reports, correlating events. LLMs can accelerate this without replacing the analyst's judgment.
Approach
Built a Python application using the Claude API to assist with threat analysis, vulnerability write-ups, and log interpretation. Includes structured prompting, context management, and output formatting for real security workflows.
